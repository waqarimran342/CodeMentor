# 🚀 CodeMentor

### *"Hot, spicy basics of C++ served daily!"*

---

## 📌 Overview

**CodeMentor** is a powerful console-based C++ learning system designed for beginners and intermediate learners.
It provides structured learning through concepts, quizzes, flashcards, and real-life examples — all in one application.

---

## ✨ Features

### 🔐 Authentication System

* User Signup & Login
* Gmail validation
* File-based user storage

### 🆓 Free Version

* 📘 Concepts (13 core C++ topics)
* 💻 Code Examples
* 🧠 Flashcards (Random facts viewer)
* 🌍 Real-life Examples
* 🔖 Bookmark system

### 💎 Premium Version

* 📝 Quiz System (3 types):

  * Code Writing Practice
  * Dry Run Practice
  * Error Analysis
* 🎮 MCQs Game with scoring system
* 📊 Score tracking
* ❓ Ask a Query (FAQs, Syntax Help, Best Practices)

### 🎯 Additional Features

* Randomized flashcards
* Input validation system
* File handling for persistent data
* Clean modular structure

## ▶️ How to Run

1. Open in any C++ IDE (CodeBlocks / VS Code / Dev C++)
2. Compile the program:

   ```
   g++ main.cpp -o app
   ```
3. Run:

   ```
   ./app
   ```
